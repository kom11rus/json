JSON - современный формат обмена данными

JSON (JavaScript Object Notation) - это легкий формат для хранения и передачи данных. Широко используется в веб-приложениях и API.

Основные преимущества

Характеристика Описание
Читаемость Простой для чтения и написания
Совместимость Поддержка всеми языками
Производительность Быстрая обработка

Базовая структура


{
    "имя": "Адам",
    "возраст": 3879,
    "адрес": {
        "город": "Райский",
        "улица": "Сад"
    },
    "теги": ["json", "data", "api"]
}
