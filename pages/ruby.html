Ruby и JSON

Стандартная библиотека


require 'json'

hash = { name: 'Alice', age: 25 }
json = hash.to_json
parsed = JSON.parse(json)

Оптимизация с Oj


require 'oj'

Oj.default_options = { mode: :compat }
json = Oj.dump(data)
parsed = Oj.load(json)
Гем Преимущества
oj ~10x быстрее
standart Базовая
